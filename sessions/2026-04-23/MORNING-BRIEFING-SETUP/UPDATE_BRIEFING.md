# How to update CMD.txt

## When a verdict arrives
1. Open CMD.txt
2. Find "VERDICTS RECEIVED" section at bottom
3. Add: [DATE] [PAPER_ID] [JOURNAL] [VERDICT]
4. Update the paper's Status in the portfolio table
5. Update "Current answer" in Daily Decision Framework

## When a new paper is submitted
1. Add row to SUBMISSION PORTFOLIO STATUS table
2. Add to BRIDGE REPO latest sessions

## Weekly (every Monday)
1. Move completed Tier 1 items to COMPLETED section
2. Promote Tier 2 items to Tier 1 if now urgent
3. Update "Last updated" date at top

## Monthly
1. Ask Claude to regenerate CMD.txt from the bridge
   repo history and current portfolio state
2. Run SICF on all papers that have not been reviewed
   this month
