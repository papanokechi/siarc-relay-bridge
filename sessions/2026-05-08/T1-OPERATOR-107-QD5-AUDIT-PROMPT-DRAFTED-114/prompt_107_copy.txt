SECTION 7 -- Forbidden-verb rules (Phase D hygiene)
==========================================================================

Forbidden verbs on prose deliverables: shows, confirms,
proves, demonstrates, establishes, verifies, verified,
validates, validated, corroborates, corroborated, certifies,
certified, settles, settled, discharges, ratifies.

Mitigation pattern: replace the verb with a noun-phrase
construction or a neutral verb that does not carry
epistemic-certification weight. Examples:
  "X confirms Y" -> "X gives Y" / "X yields Y" / "X
                    indicates Y" / "X holds Y"
  "X verifies Y" -> "X computes Y at <dps> digits" /
                    "X reaches Y" / "X locates Y"
  "X discharges Z" -> "X addresses Z" / "X resolves Z"
  "X establishes Y" -> "X determines Y" / "X gives Y"

098-J3 EXEMPT categories (residual hits in these are
NOT halts):
  - verb-list-as-data inside JSON quoted strings (e.g.,
    halt regex declarations like
    "trigger":"forbidden-verb scan returns >0 hits")
  - verbatim quotations from substrate files enclosed in
    blockquote (>) syntax or fenced code blocks (```)
  - SHA / filesystem-inspection telemetry
  - finding-summary references (e.g., "the 113 verdict
    finding")
