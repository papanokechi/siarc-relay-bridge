OKAMOTO 1987 S1 INTRODUCTION (verbatim from 110 prestage)
================================================================

The present article concerns the third Painleve equation:

P_III:  d^2 q / dt^2  =  (1/q)(dq/dt)^2  -  (1/t)(dq/dt)
                       +  (1/t)(alpha q^2 + beta)
                       +  gamma q^3  +  delta / q,

for which we make the assumption gamma delta != 0, throughout
this paper.

This equation is well-known and investigated by many authors
(for example, [3], [5], [6], [7]), while we consider in the
following mainly the equation:

P_III':  d^2 q / dt^2  =  (1/q)(dq/dt)^2  -  (1/t)(dq/dt)
                        +  q^2/(4 t^2)(gamma q + alpha)
                        +  beta/(4 t)  +  delta/(4 q),

instead. These two equations are equivalent each other. In fact,
by replacing in P_III, t^2 by t and q by t q, we obtain P_III'.

================================================================
OKAMOTO 1987 H_III HAMILTONIAN (verbatim)
================================================================

The Hamiltonian associated with P_III is:

H_III  =  (1/t)  [  2 q^2 p^2
                   -  { 2 eta_inf t q^2  +  (2 theta_0 + 1) q
                        -  2 eta_0 t  }  p
                   +  eta_inf (theta_0 + theta_inf) t q  ],

where the constants eta_Delta, theta_Delta (Delta in {0, inf})
are connected to alpha, beta, gamma, delta of the equation as
follows:

(0.1)
   alpha  =  -4 eta_inf theta_inf,
   beta   =   4 eta_0 (theta_0 + 1),
   gamma  =   4 eta_inf^2,
   delta  =  -4 eta_0^2.

By the assumption [gamma delta != 0], we have eta_Delta != 0;
moreover we set eta_Delta = 1 without loss of generality.

NOTE (109-O1 = STANDING ASSUMPTION ANCHOR): Okamoto's S1
standing assumption eta_Delta != 0 holds throughout the paper.
The WLOG normalisation eta_inf = eta_0 = 1 is imposed for
canonical-form work; outside the WLOG, the H_III Hamiltonian
retains the eta_Delta dependence above.

================================================================
OKAMOTO 1987 H_III' HAMILTONIAN (verbatim)
================================================================

On the other hand, the Hamiltonian associated with P_III' is:

H_III'  =  (1/t)  [  q^2 p^2
                    -  { eta_inf q^2  +  theta_0 q
                         -  eta_0 t  }  p
                    +  (1/2) eta_inf (theta_0 + theta_inf) q  ].

These two Hamiltonians are connected mutually through the
canonical transformation phi:

(0.2)
   q  ->  t q,    p  ->  t^{-1} p,    t  ->  t^2,
   H_III'  ->  (1/(2 t))(H_III + (1/t) q p).

NOTE (109-O2 = THE FW (4.1) IS FW PIII' WITH OKAMOTO WLOG):
expanding Okamoto's t H_III' (multiplying the (1/t) prefactor
through):

  t H_III'  =  q^2 p^2  -  eta_inf q^2 p  -  theta_0 q p
                       +  eta_0 t p  +  (1/2) eta_inf (theta_0 + theta_inf) q

Compare with FW (4.1):

  tH  =  q^2 p^2  -  q^2 p  -  v_1 q p  +  t p  +  (1/2)(v_1 + v_2) q

Match coefficient-by-coefficient:
  coeff of q^2 p:   -eta_inf vs. -1   =>  eta_inf = 1
  coeff of q p:     -theta_0 vs. -v_1 =>  v_1 = theta_0
  coeff of t p:      eta_0 vs. 1      =>  eta_0 = 1
  coeff of q:        (1/2) eta_inf (theta_0 + theta_inf)
                  vs. (1/2)(v_1 + v_2)
                  =>  eta_inf (theta_0 + theta_inf) = v_1 + v_2
                  =>  with eta_inf = 1:  theta_0 + theta_inf = v_1 + v_2
                  =>  with v_1 = theta_0:  theta_inf = v_2

Therefore the literature-anchored identification (UF-110-3) is:

  v_1  =  theta_0     (under WLOG eta_inf = eta_0 = 1)
  v_2  =  theta_inf   (under WLOG eta_inf = eta_0 = 1)

ANY DEPARTURE FROM OKAMOTO'S WLOG (eta_inf != 1 OR eta_0 != 1)
breaks the FW (4.1) match: FW (4.1) does NOT contain free
eta_inf or eta_0 parameters; eta_inf = 1 is hardcoded in the
"-q^2 p" coefficient and eta_0 = 1 is hardcoded in the "+t p"
coefficient.
