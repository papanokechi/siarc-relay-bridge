SECTION 2.1 -- THE JIMBO-MIWA-OKAMOTO SIGMA-FORM OF PV
(verbatim, .txt L1740-1855)
================================================================

L1740: 2.1 The Jimbo-Miwa-Okamoto sigma-form of PV
L1741: As formulated in [30], the Okamoto tau-function theory of the fifth
L1742: Painleve equation PV is based on the Hamiltonian system {Q, P, t, K}
L1743: (2.1)
L1748:   tK = Q(Q-1)^2 P^2 - [(v_2 - v_1)(Q-1)^2 - 2(v_1 + v_2) Q(Q-1) + tQ] P
L1749:        + (v_3 - v_1)(v_4 - v_1)(Q - 1)
L1750:
L1751: where the parameters v_1, ..., v_4 are constrained by
L1752: (2.2)
L1754:    v_1 + v_2 + v_3 + v_4 = 0.

NOTE (109-A1): equation (2.2) is the FW PV NULL-SUM CONSTRAINT.
The four parameters v_1, ..., v_4 are constrained to a 3-dimensional
linear subspace of R^4 (or C^4) by this single linear equation.

L1798: (2.5)
L1801:   alpha = (1/2)(v_3 - v_4)^2,
L1804:   beta  = -(1/2)(v_2 - v_1)^2,
L1808:   gamma = 2 v_1 + 2 v_2 - 1,
L1812:   delta = -1/2.

NOTE (109-A2): the PV ODE coefficients (alpha, beta, gamma, delta)
of (2.5) involve all four (v_1, v_2, v_3, v_4); but at the
delta = -1/2 normalisation only differences of the v_i (and the
sum v_1 + v_2 in gamma) appear in the ODE. This is consistent with
(2.2) reducing four to three free parameters.

================================================================

L1835: Proposition 2.1 [21, 30] With K specified by (2.1), define the
L1836: auxiliary Hamiltonian h by
L1837: (2.7)
L1839:   h = tK + (v_3 - v_1)(v_4 - v_1) - v_1 t - 2 v_1^2.
L1840:
L1841: The auxiliary Hamiltonian h satisfies the differential equation
L1842: (2.8)
L1846:   (t h'')^2 - [h - t h' + 2(h')^2]^2
L1850:    + 4 (h' + v_1)(h' + v_2)(h' + v_3)(h' + v_4) = 0.

NOTE (109-A3): the FW (2.7) auxiliary Hamiltonian for PV has
additive shift (v_3 - v_1)(v_4 - v_1) (a c-number quadratic in
the v_i)  +  (-v_1 t) (a t-linear shift)  +  (-2 v_1^2) (a c-number
quadratic shift). This is the PV-side analogue of the PIII (4.3)
auxiliary Hamiltonian in mechanism (a).

L3903: 4 tau-function theory of PIII and hard edge scaling
L3904:
L3905: In the Laguerre ensemble the eigenvalues are restricted to be
L3906: positive. For N large, the spacing between the eigenvalues in the
L3907: neighbourhood of the origin is of order 1/N. By scaling the
L3908: coordinates as in (1.54) the eigenvalue spacing is then of order
L3909: 1 and well defined distributions result. In this limit the PV
L3910: system degenerates to the PIII system so it is appropriate to
L3911: revise the tau-function theory of the latter.

NOTE (109-A4): the FW PV->PIII "degeneration" is described in

NOTE (109-A4b): FW S4.1 just *postulates* the PIII Hamiltonian

PROPOSITION 4.1 -- FW (4.3) AUXILIARY HAMILTONIAN h
(verbatim, .txt L3956-4045)
================================================================

L3956: Proposition 4.1 [31] With H specified by (4.1), define the
L3957: auxiliary Hamiltonian
L3958: (4.3)
L3960:   h = tH + (1/4) v_1^2 - (1/2) t.

NOTE (109-A7 = THIS IS THE CENTRAL EQUATION FOR MECHANISM (a)):
the FW (4.3) auxiliary Hamiltonian shift consists of:
  - quadratic c-number shift: (1/4) v_1^2  (depends on v_1 only)
  - t-linear shift:           -(1/2) t     (independent of v_1, v_2)
  - no v_2-dependent shift
  - no constant-in-t shift involving v_2

L3966: The auxiliary Hamiltonian h satisfies the differential equation
L3967: (4.4)
L3970:   (t h'')^2 + v_1 v_2 h' - (4 (h')^2 - 1)(h - t h')
L3971:    - (1/4)(v_1^2 + v_2^2) = 0.

NOTE (109-A8): proof outline at L3973-4010 uses Hamilton equations
(2.3), substitutes v_1 v_2 / 2 cross-term in the proof; the proof
itself only requires the surviving (v_1, v_2) pair.

L4011: Of interest in the random matrix application is the variant of
L4012: (4.4) satisfied by
L4013: (4.6)
L4015:   sigma_III(t) := -(tH)|_{t -> t/4}  -  (t/4)(v_1 - v_2)  +  v_1 / 4.

NOTE (109-A9): FW (4.6) sigma_III differs from (4.3) h by an
additional  -(t/4)(v_1 - v_2) + v_1/4  shift (and a t -> t/4
rescaling in tH). Mechanism (a) per prompt 105 + 069r2 named
specifically the (4.3) auxiliary Hamiltonian, not the (4.6)
sigma_III variant; we test (4.3) here. The (4.6) variant is a
separate substrate slot for any future cascade refinement.
