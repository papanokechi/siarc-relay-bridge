# Mathematics of Computation — Submission Requirements (snapshot)
# Source URL: https://www.ams.org/publications/journals/journalsframework/mcomsubmit
# Fetched: 2026-05-06
# Fetcher: GitHub Copilot (VS Code) for relay 046

Single anonymous peer review (reviewers know authors; authors do not know reviewers).
Centralized Manuscript Processing initial submission via PDF upload at
  https://ams.msp.org/submit_new.php?j=mcom

Author limit: 3 submissions per 12-month rolling window per author.

Required with submission:
  - Journal name
  - Corresponding author name + email + mailing address
  - **Suggested Editor** from the Math.Comp. editorial board (NOT a list of
    suggested referees — Math.Comp. uses an editor-suggestion model, not
    a referee-suggestion model)
  - 2020 Mathematics Subject Classification (primary + secondary), placed
    in footnotes
  - Abstract ≤ 300 words

No prior or concurrent submission elsewhere (standard exclusivity).

After Acceptance: Consent to Publish + Copyright agreement; if Open Access,
choose Creative Commons license.

Useful tools:
  - AMS journal article templates (Overleaf): https://www.overleaf.com/org/ams/
  - AMS-prepared style files: https://www.ams.org/mcom/mcomauthorpac.html

NOTE FOR RELAY-046 COVER LETTER:
The relay prompt §3 specifies ¶5 = "Suggested referees: 3 names". This is
the JTNB norm. Math.Comp. asks for a suggested *editor* from their board
(see https://www.ams.org/mcomedit). The drafted cover letter follows the
relay prompt's structure (3 referees) BUT the operator should at submission
time replace ¶5 with a suggested-editor section pointing to a Math.Comp.
editorial-board member whose interests cover PCF / experimental number
theory / PSLQ algorithmics. Candidate editors are listed in
operator_dispatch_checklist.md as a side-channel reminder.
